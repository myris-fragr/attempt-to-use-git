## なぜgit？

gitはテキストを扱う上でめっちゃ便利です。

- 修正前と後の差分がわかる
- 変更履歴がわかる
- 世界線に分岐を作って（！）違うバージョンを管理できる
- 分岐をあとからまとめることができる

また、Githubなどのサービスを利用してリモートの環境を作ればこんな事もできます。

- 複数の端末から編集できる
- 問題点を洗い出して一つずつ解決していける
- チーム作業ができる
